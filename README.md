# Available Scripts

- Clone the repo

  ```bash
  $ git clone https://github.com/francislagares/krypt-blockchain-app.git
  ```

- Install the dependencies by running the following command.

  ```bash
  yarn install
  ```

- Start the development server:

  ```bash
  yarn start
  ```

- Start the test suites:

  ```bash
  yarn test
  ```
